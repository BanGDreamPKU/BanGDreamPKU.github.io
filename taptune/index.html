🎬 TapTuneTap-Track Alignment Tool

音游「手元」对轴配乐工具——把只有敲击音的手元录像，自动配上对齐到轴的原曲。

录手元时游戏内音乐往往是静音的，只留下清脆的敲击音效。TapTune 会自动认出这是哪首歌、拉取谱面与原曲，用敲击音与谱面对轴求出偏移，再把原曲精确贴回、与（可调音量的）敲击音混合，导出成片。

支持多款音游，目前内置三个完整后端：

  • BanG Dream! · Bestdori
  • Project Sekai · sekai.best
  • World Dai Star

怎么用

  1. 录一段手元：游戏内音乐静音，只保留敲击音效。
  2. 下载解压后运行 taptune.exe，在打开的界面里选择这段视频。
  3. 等待自动认歌、对轴、配乐，导出成片即可。

也提供命令行：taptune identify <视频> 认歌、taptune export <视频> <成片.mp4> 一键出片。

原理

认歌（画面 OCR） → 拉谱面 / 原曲 → 敲击音 × 谱面对轴求偏移 τ → 原曲精确贴回 + 混音 → 导出

谱面的 note 时间是真值，录音只用来估一个全局偏移；原曲落位用视频与原曲的包络互相关，避免节拍网格的整拍歧义，也能识别倍速录像。

倍速录像

用 Sonolus #SPEED 滑块整体加减速（0.5–2.0x）打出的手元也能处理：常速对轴失败后自动扫倍速网格，识别到倍速就把原曲保音高拉伸到同一速率再对位出片，全程无需手动设置。录屏帧率差（59.94 / 60fps）造成的 0.1% 级整体漂移也会一并自动补偿。

已知限制

  • 曲内变速谱：谱面在一首歌内速率随时间变化（不是整首统一倍速），无法用单一偏移对齐，会被标记低置信度，需人工核对。整首统一的倍速录像不受此限，见上。
  • 敲击音被伴奏严重污染的录像：置信度下降，但通常仍能锁定偏移。
  • 出片需联网，用于拉取谱面与原曲音源。
